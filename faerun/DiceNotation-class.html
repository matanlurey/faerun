DiceNotation

DiceNotation class final

A notation for dice expressions.

Dice expressions are strings that represent a combination of dice rolls and arithmetic operations.

For example, 2d6 + 3 represents the sum of rolling two 6-sided dice and adding 3, and can be represented as:

  • 2 * Dice.d6 + 3
  • DiceNotation(Dice.d6, amount: 2, modifier: 3)
  • DiceNotation.parse('2d6 + 3')

Equality

Two dice notations are considered equal if they have the same dice, amount, and modifier.
