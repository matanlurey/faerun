Action

Action class final

Describes doing something other than moving or communicating.

main lists the game's main actions; more can be defined with Action.new:

Action('Climb');

Actions are intended to be used as a reference for what a character can do during their turn. They are not typically stored as part of a character's state but might persist in a log of actions taken during a game.

Equality

Actions are compared based on their name.
